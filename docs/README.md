# Active Docs - SPHERE II Entrepreneurship

This folder is for active entrepreneurship work only.

Use `docs/` for:

- venture and opportunity maps
- validation research guides
- ecosystem and market signal registries
- public case skeletons
- founder operating templates
- GPT-assisted venture research packs

Historical or superseded material should move to:

- `archived_docs/`

## Folder structure

- `docs/core/`
  - cross-sphere entrepreneurship execution notes
  - structure approval gate
  - E1-E4 research execution index
  - GPT-assisted venture research workflow

## Current active logic

SPHERE II should not become generic startup content.

It should answer:

- what opportunity is being explored
- what evidence would validate or weaken it
- what market, customer, ecosystem, or operating signal matters
- what practical next step a founder, mentor, or builder can take

## Before real research execution

Use `docs/core/ENTREPRENEURSHIP_STRUCTURE_APPROVAL_GATE.md` before turning a venture topic into a research artifact.

Use `docs/core/E1_E4_RESEARCH_EXECUTION_INDEX.md` as the cross-lane index for deciding which entrepreneurship route to execute next.

Use `docs/core/GPT_ASSISTED_ENTREPRENEURSHIP_WORKFLOW.md` when a venture research pack is being sent into GPT and returned as repo-ready `.md` output.
