# Practice Tasks · SPHERE II / ENTREPRENEURSHIP

## Starter Tasks

### `E1` · Venture / Product / Opportunity

1. Turn one vague startup idea into a structured opportunity map.
2. Write a one-page problem statement using a real domain example.
3. Compare `good idea` vs `good problem` vs `good market timing`.

### `E2` · Validation / GTM / Market

1. Build a first-pass validation checklist for a chosen idea.
2. Draft five customer interview questions that are actually useful.
3. Map one startup idea into `audience -> need -> message -> first channel`.

### `E3` · Ecosystem / Experts / Partners

1. Build a mini ecosystem map around one direction.
2. Add one useful startup, expert, or ally to the support registry.
3. Draft a `who should I talk to first?` note for a founder.

### `E4` · Public Cases / Founder Ops

1. Write a weekly founder update template.
2. Convert one founder story into a public case structure.
3. Create a lean `build / wait / kill` decision board for an MVP idea.
